Face Morphing Algorithm ¿Cómo funciona?

Este algoritmo extrae automáticamente los puntos característicos del rostro y en función de estos puntos, las imágenes se dividen y se realiza la transformación. El algoritmo se ha utilizado para generar la transformación entre imágenes de rostros de diferentes personas y los resultados de esta transformación interpersonal e intrapersonal son subjetivamente satisfactorios.

El reconocimiento de rostros, se asocia al campo de la Inteligencia Artificial, debido a que intenta resolver un tipo problema en el que los seres humanos son más rápidos y precisos.

¿Cómo implementarlo?

Especificaciones Proyecto Requerimientos Script: Locator Script: Aligner Script: Warper Script: Morpher

Especificaciones técnicas

Desarrollado en Python, dlib, Numpy, Scipy, dlib.

Compatible con Python 2.7, Python 3.6+

Probado en Windows, MacOS y Linux 64bit (dockerizado).

Requerimientos

Ejecutar en la consola: pip install -r requirements.txt

Descargar: http://dlib.net/files/shape_predictor_68_face_landmarks.dat.bz2 y extraer el archivo.

Exportar variable de entorno: DLIB_DATA_DIR donde el shape_predictor_68_face_landmarks.dat es referenciado.
Comando: export DLIB_DATA_DIR=/Downloads/data

Clonar proyecto

Usar como línea de comando:

$ git clone https://github.com/ISABELUNA16/facemorphing_test.git

Learn More

Locator.py

- Localiza puntos faciales.

- Para un localizador diferente, devuelva una matriz de puntos de control (x, y).

Aligner.py

- Alinea las rostros cambiando el tamaño, centrándolo y recortándolo.

Warper.py

- Dadas 2 imágenes y sus puntos faciales, deforma una imagen a la otra.

- Triangula los puntos faciales.

- Transforma cada triángulo con interpolación bilineal.

Morpher

- 4a. Morpher: Morph entre 2 o más imágenes.

- 4b. Averager: Caras promedio de 2 o más imágenes.

Ejemplos

Creando un video morphing entre las 2 imágenes.

Martín Vizcarra (Presidente del Perú)

1.- Trazar secuencia de imágenes:

- python facemorpher/morpher.py --src=isabel.jpg --dest=vizcarra.jpg --out_frames=out_folder_isabel_vizcarra --num=24 --plot

2.- Creando Morphing:

- python facemorpher/morpher.py --src=isabel.jpg --dest=vizcarra.jpg --out_frames=out_folder_isabel_vizacarra --out_video=out_isabel_vizcarra.avi --num=100
